# LotionGPT

Modern Chat UI powered by Z.ai GLM-4.6 with multimodal support.

## Features

- Modern & Sleek Design mit Glasmorphismus
- Chat-Historie mit Search & Filter (Alle/Heute/Woche)
- Multimodal Support (Text + Bilder)
- Message Actions (Copy, Regenerate)
- Toast Notifications
- Responsive Design (Mobile, Tablet, Desktop)
- Smooth Animations & Micro-interactions

## Tech Stack

- Next.js 16 (App Router)
- React 19
- TypeScript
- Tailwind CSS 4
- Z.ai GLM-4.6 API

## Local Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Deployment

### Vercel

1. Push to GitHub
2. Import repository in Vercel
3. Deploy

## Environment Variables

Create `.env.local`:

```
ZAI_API_KEY=your_api_key_here
```
